Electronic Version of the Book


The SETUP\EBOOK\ALPHA32 directory contains an electronic version of the book as a compiled HTML Help file. This file is named PRGAPPS4.CHM.

In order to view this electronic book you must have the proper software installed. The following sofware is required:
  • Microsoft Internet Explorer 4.0 or higher
  • HTML Help components 1.2 or higher
To download the latest version of Internet Explorer, consult Microsoft's Internet Explorer Web site:
http://www.microsoft.com/ie/

If you try to view the electronic book and a message is displayed indicating that a newer version of the HTML Help ActiveX Control (HHCTRL.OCX) is needed to view the file, run the HTML Help update (HHUPD.EXE) to update your HTML Help components. For information about HTML Help and to download the latest version, consult Microsoft's HTML Help Website:
http://msdn.microsoft.com/workshop/author/htmlhelp/
